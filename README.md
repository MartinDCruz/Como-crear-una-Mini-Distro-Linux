# 🐧 Código Oficial: Cómo crear una Mini-Distro Linux

Bienvenido al repositorio oficial del libro **"Cómo crear una Mini-Distro Linux: Sin interfaces gráficas ni bloatware"** por Martin D. Cruz.

Este espacio ha sido diseñado para acompañar tu lectura, proporcionando los scripts, archivos de configuración y comandos exactos necesarios para construir un sistema operativo puro desde el nivel más bajo.

## 🚀 Cómo utilizar este código

El código está organizado por capítulos para que coincida con el progreso de tu lectura. Para utilizarlo en tu entorno de desarrollo, puedes clonar el repositorio completo:

`git clone https://github.com/MartinDCruz/Como-crear-una-Mini-Distro-Linux.git`

Si prefieres trabajar con archivos individuales, puedes navegar por las carpetas arriba y utilizar el botón "Copy raw contents" para pegar el código directamente en tu terminal o editor de texto.

## ⚠️ Aviso Legal e Ingeniería Defensiva

Como se detalla en el manual, las herramientas de bajo nivel utilizadas en este proyecto (`dd`, `fdisk`, `sfdisk`, `mkfs`) son intrínsecamente destructivas. 

**Este código debe ejecutarse exclusivamente en entornos de laboratorio controlados o máquinas virtuales (como QEMU).** El autor no se hace responsable de la pérdida de datos en hardware físico por el uso incorrecto de estos scripts. Siempre verifica a qué bloque de dispositivo apunta tu comando antes de presionar Enter.
